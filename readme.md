# Interactive AI Music Prompt Skill
# 交互式 AI 音乐 Prompt Skill

> From vague music ideas to usable AI music prompts.  
> 从模糊灵感，到可直接拿去生成音乐的 Prompt 包。

一个面向 AI 音乐创作的本地 skill。  
它帮助你把主题、情绪、场景、风格偏好，快速整理成**结构清晰、可直接复制、可继续迭代优化**的音乐 Prompt。

---

## 为什么你可能会喜欢它

很多人做 AI 音乐时，卡住的并不是“没有想法”，而是：

- 脑子里有画面，但不知道怎么描述给 AI
- 想做一首歌，但只知道“差不多这个感觉”
- 想做 BGM，但不知道怎么控制结构、时长和推进
- 已经有歌词或 Prompt，但总觉得不够像歌、不够准、不够抓耳

这个 skill 的目标，就是把这些模糊表达，变成一份真正可执行的音乐 Prompt 包。

---

## 它能帮你做什么

### 创作带歌词歌曲 Prompt
适合主题歌、情绪表达歌、短视频传播歌、城市主题歌、毕业歌、励志歌等。

### 创作纯器乐 / BGM Prompt
适合短视频配乐、Vlog 背景音乐、产品视频配乐、空间氛围音乐、治愈轻音乐等。

### 优化已有歌词或 Prompt
适合你已经写了一版，但希望：

- 副歌更抓耳
- 情绪更集中
- 风格更清晰
- 更适合传播
- 更适合某个使用场景

### 生成多个创作方向
同一个主题，可以帮你拆成多个可比较的音乐方向，而不是只给一个答案。

---

## 这个 skill 的体验特点

### 1. 不走冗长问卷
它不会一上来连问十几个问题。  
默认会先利用你已经给出的信息，只在真的影响结果质量时才补问关键问题。

### 2. 歌曲和 BGM 分开处理
它不会把“写歌”和“做配乐”混成一套逻辑。

它会尽早区分：

- 带歌词歌曲
- 纯器乐 / BGM

然后分别走不同的生成路径。

### 3. 输出不是一句话，而是一整包
默认不是只给一句泛泛描述，而是直接给你**完整 Prompt 包**。

### 4. Style Prompt 更容易复制和改写
风格部分不是一整段很难改的长英文，而是结构化字段格式，方便你自己继续微调。

### 5. 宽泛主题会继续帮你收敛
如果你只说“杭州”“夏天”“毕业”“父亲”这种大主题，它不会草率脑补，而是继续确认你真正想表达的方向。

## 默认会输出什么

### 如果你要的是歌曲
通常会输出：

- 需求摘要
- 音乐方向
- 推荐歌名
- Style Prompt
- Lyrics
- 下一步优化方向
- 使用建议

### 如果你要的是纯器乐 / BGM
通常会输出：

- 需求摘要
- 音乐方向
- Style Prompt
- Instrumental Structure Box
- 下一步优化方向
- 使用建议

默认直接输出**完整 Prompt 包**。  
除非你明确要求简洁版、歌词版、多版本等特殊格式，否则不会额外追问输出形式。

## 你可以怎么用它

### 示例 1：写歌
- 创作一首中文流行歌曲，主题是毕业，温暖怀旧，但最后有一点希望感
- 写一首中文 bossanova 歌曲，在炎炎夏日来一点舒爽的感觉
- 创作一首描写上海夜生活的中文 Rap，男女对唱，对话感更强

### 示例 2：做 BGM
- 创作一段疗愈的轻音乐，适合睡前放松，60–90 秒，循环优先
- 做一段适合盛夏咖啡馆的爵士钢琴，清新舒爽
- 做一段科技产品发布会用的高级感 BGM，30–60 秒，旁白友好

### 示例 3：优化现有内容
- 帮我优化这段歌词，让副歌更抓耳
- 帮我把这段音乐 Prompt 改得更适合短视频传播

## 推荐输入方式

如果你希望更快得到高质量结果，推荐直接这样说：

### 歌曲创作
- 写一首【语言】的【风格】歌曲，主题是【主题】，想表达【内容 / 情绪】，适合【场景】

例如：
- 写一首中文流行歌曲，主题是毕业，想表达温暖怀旧但最后有一点希望感，适合发短视频和朋友圈

### 纯器乐 / BGM
- 创作一段【风格】的纯音乐，给人【感受】，适合【场景】，时长【长度】，优先【循环 / 剪辑】

例如：
- 创作一段疗愈的轻音乐，给人慢慢放松下来的感觉，适合睡前聆听，时长 60–90 秒，优先循环

### 优化已有内容
- 这是我现在的【歌词 / Prompt】，我希望它变得更【目标】

例如：
- 这是我现在的歌词，我希望副歌更抓耳、更适合传播，男女对唱更明显

### 参考方向型输入
- 我想做一首【类型】，我喜欢【某首歌 / 某位歌手】那种【风格特征】，但不要直接模仿

例如：
- 我想做一首世界杯主题歌，我喜欢《意大利之夏》那种大开阔、史诗感、希望感很强的方向，但不要直接模仿

## 输出长什么样

### Style Prompt 会更结构化

```text
Genre: Chinese folk, urban narrative folk
Mood: gentle, nostalgic, relaxed
Vocal: natural sincere male lead
Tempo: mid-slow, steady storytelling pace
Texture: acoustic guitar, light percussion, soft bass
Energy: calm entrance, gradual unfolding, soft return
Hook: natural and memorable chorus
Use Case: personal sharing / city memory expression
```

### 歌词区不只是歌词
它通常也会承担：

- 段落标签
- 编曲控制
- 演唱角色分配

例如：

```text
[Verse 1 - acoustic guitar, close vocal]
...

[Chorus - fuller harmony, stronger lift]
...
```

### 纯器乐也有结构控制
纯器乐不会只给一个风格词，还会给 `Instrumental Structure Box`，用于控制：

- 时长
- 段落推进
- 循环方式
- 剪辑点
- 结构变化

---

## 它不会做什么

这个 skill 不会：

- 直接模仿真实歌手或现有歌曲
- 输出复制型“像某某一样”的 Prompt
- 把音乐创作做成死板问卷

如果你给了歌手或作品参考，它会提炼：

- 风格特征
- 情绪方向
- 编曲质感
- 演唱气质

而不会直接模仿。

## 常见错误输入

### 主题太宽
- 杭州
- 夏天
- 父亲

更好的写法：
- 写一首关于杭州的民谣，想表达一个人在这座城里慢慢生活下来的感觉

### 只有风格，没有内容
- 来一首 Rap
- 做一段民谣

更好的写法：
- 写一首描写上海夜色与都市节奏的中文 Rap

### 只有情绪，没有场景
- 做一段疗愈的轻音乐

更好的写法：
- 做一段疗愈的轻音乐，适合睡前放松，循环播放

### 直接要求模仿歌手
- 写一首像 Beyond 那样的歌

更好的写法：
- 想要真诚热血、乐队感强、理想主义气质的励志摇滚歌

### BGM 没有时长
- 做一段科技感 BGM

更好的写法：
- 做一段科技感 BGM，适合产品发布视频，30–60 秒，旁白友好

## 安装方式

这是一个**本地 Skill 文件夹**。

### 最小要求

- 根目录存在 `SKILL.md`
- `SKILL.md` 引用到的文件完整保留
- 你的宿主环境支持加载本地 skill

### 推荐安装步骤

1. 把整个 skill 文件夹放入你的本地 skills 目录
2. 保持目录结构不变
3. 确保以下内容都在：
   - `SKILL.md`
   - `agents/openai.yaml`
   - `references/`
   - 相关 `ai_music_*.md` 规则文件
4. 重启或刷新你的 skill 运行环境
