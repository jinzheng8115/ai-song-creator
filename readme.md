# Interactive AI Music Prompt Skill
# 交互式 AI 音乐 Prompt Skill

把一个模糊的音乐想法，快速变成一份**可直接使用、可继续优化、结构清晰的 AI 音乐 Prompt 包**。

Whether you want to write a song, generate a BGM, refine lyrics, or explore multiple directions, this skill helps you go from idea to usable prompt faster.
无论你是想写一首歌、做一段 BGM、优化歌词，还是对比多个创作方向，这个 skill 都能帮你更快把想法变成可用的 Prompt。

---

## 适合谁用

这个 skill 适合你，如果你经常遇到这些情况：

- 想写歌，但脑子里只有一个主题或画面
- 想做 BGM，但不知道怎么描述给 AI 音乐工具
- 已经有歌词或 Prompt，但总觉得不够准
- 想要几个不同方向对比一下，再决定继续往哪走
- 想做一首适合短视频、朋友圈、品牌内容或个人表达的音乐

---

## 它能做什么

### 1. 创作歌曲 Prompt
输入一个主题、风格、场景或情绪，它会帮你整理成完整歌曲 Prompt 包。

例如：
- 写一首中文 bossanova 歌曲，在炎炎夏日来一点舒爽的感觉
- 创作一首描写上海夜色的中文 Rap 歌曲
- 写一首关于杭州的中文民谣

### 2. 创作纯器乐 / BGM Prompt
如果你要的是没有歌词的音乐，它会帮你输出适合 AI 音乐工具的器乐 Prompt。

例如：
- 做一段疗愈的轻音乐，适合睡前放松
- 来一段适合盛夏咖啡馆的爵士钢琴
- 做一段科技产品发布会用的高级感 BGM

### 3. 优化已有歌词或 Prompt
如果你已经写了一些内容，但总觉得“不够像歌”或“不够像那个风格”，它也可以继续帮你优化。

## 使用方式很简单

你只需要像聊天一样输入需求。

例如：

- 写一首中文 bossanova 歌曲，在炎炎夏日来一点舒爽的感觉
- 创作一段疗愈的轻音乐，适合睡前放松
- 写一首描写上海的歌，想做成叙事感 Rap

如果你的信息还不够，它会只补问少量关键问题，例如：

- 这首歌是用在什么场景？
- 你真正想表达的内容是什么？
- 是男声、女声、还是对唱？
- BGM 需要多长？是循环优先还是剪辑优先？

默认会尽量少问，不走冗长问卷路线。

---

## 最快上手：直接套这几种输入句式

如果你希望第一次就更快得到高质量结果，可以直接套下面这些句式：

### 句式 1：歌曲创作
- 写一首【语言】的【风格】歌曲，主题是【主题】，想表达【内容 / 情绪】，适合【场景】

例如：
- 写一首中文流行歌曲，主题是毕业，想表达温暖怀旧但最后有一点希望感，适合发短视频和朋友圈

### 句式 2：纯器乐 / BGM
- 创作一段【风格】的纯音乐，给人【感受】，适合【场景】，时长【长度】，优先【循环 / 剪辑】

例如：
- 创作一段疗愈的轻音乐，给人慢慢放松下来的感觉，适合睡前聆听，时长 60–90 秒，优先循环

### 句式 3：优化已有内容
- 这是我现在的【歌词 / Prompt】，我希望它变得更【目标】

例如：
- 这是我现在的歌词，我希望副歌更抓耳、更适合传播，男女对唱更明显

### 句式 4：参考方向型输入
- 我想做一首【类型】，我喜欢【某首歌 / 某位歌手】那种【风格特征】，但不要直接模仿

例如：
- 我想做一首世界杯主题歌，我喜欢《意大利之夏》那种大开阔、史诗感、希望感很强的方向，但不要直接模仿

## 默认会输出什么

### 歌曲类默认输出
- 需求摘要
- 音乐方向
- 推荐歌名
- Style Prompt
- Lyrics
- 下一步优化方向
- 使用建议

### 纯器乐 / BGM 默认输出
- 需求摘要
- 音乐方向
- Style Prompt
- Instrumental Structure Box
- 下一步优化方向
- 使用建议

默认直接输出**完整 Prompt 包**。

除非你明确要求简洁版、多版本、只要歌词等特殊形式，否则不会额外追问你“要哪种输出格式”。

---

## 输出有什么特点

### 1. Style Prompt 更易读
Style Prompt 不是一大段难读的英文，而是结构化字段格式，方便阅读、复制和微调。

例如：

```text
Genre: Chinese folk, urban narrative folk
Mood: gentle, nostalgic, relaxed
Vocal: natural sincere male lead
Tempo: mid-slow, steady storytelling pace
Texture: acoustic guitar, light percussion, soft bass
Energy: calm entrance, gradual unfolding, soft return
Hook: natural and memorable chorus
Use Case: personal sharing / city memory expression
```

### 2. 歌词不仅是歌词
对于歌曲类输出，`Lyrics` 不只是纯正文，还会承载：

- 段落标签
- 编曲控制
- 角色分配（如果有）

例如：

```text
[Verse 1 - acoustic guitar, close vocal]
...

[Chorus - fuller harmony, stronger lift]
...
```

### 3. 纯器乐也有结构控制
纯器乐虽然没有歌词，但仍然会有：

- `Instrumental Structure Box`

它负责控制：

- 乐曲长度
- 段落推进
- 循环方式
- 剪辑点
- 结构变化

所以它不只是给你一个“风格”，而是给你一段真正可控的器乐生成 Prompt。

---

## 它怎么处理“主题太宽”的情况
如果你只给了一个很大的主题，比如：

- 杭州
- 上海
- 夏天
- 父亲
- 校园

它不会马上脑补成一首歌。

它会继续问你：

- 你到底想表达什么？
- 是城市气质、个人记忆、爱情相遇、漂泊感，还是治愈和停留？

这样生成出来的内容会更贴近你的真实意图，而不是流于表面。

---

## 它不会做什么

这个 skill 不会：

- 直接模仿真实歌手或现有歌曲
- 生成“像某某一样”的复制型 Prompt
- 把 AI 音乐创作做成一份死板问卷

如果你给了参考歌手或歌曲，它会提炼：

- 风格特征
- 情绪方向
- 编曲质感
- 演唱气质

而不会直接输出模仿型内容。

## 常见错误输入示例

下面这些输入不是不能用，但通常会让 skill 多追问几轮，或者更容易输出泛化结果。

### 常见错误 1：只有一个宽泛主题
- 杭州
- 夏天
- 父亲

问题：
- 主题太大
- 不知道你到底想表达城市气质、个人记忆、爱情相遇，还是漂泊感

更好的写法：
- 写一首关于杭州的民谣，想表达一个人在这座城里慢慢生活下来的感觉

### 常见错误 2：只有风格，没有内容
- 来一首 Rap
- 做一段民谣

问题：
- 只有风格，没有主题、情绪或场景

更好的写法：
- 写一首描写上海夜色与都市节奏的中文 Rap

### 常见错误 3：只有情绪，没有使用场景
- 做一段疗愈的轻音乐

问题：
- 疗愈可以是睡前、学习、冥想、空间氛围，结果差别很大

更好的写法：
- 做一段疗愈的轻音乐，适合睡前放松，循环播放

### 常见错误 4：只给参考歌手或作品名
- 写一首像 Beyond 那样的歌
- 来一首像某某歌手风格的歌

问题：
- 这会触发直接模仿问题
- skill 不会按具体歌手做复制型输出

更好的写法：
- 想要真诚热血、乐队感强、理想主义气质的励志摇滚歌

### 常见错误 5：对纯器乐没有时长要求
- 做一段科技感 BGM

问题：
- 纯器乐时长和结构会直接影响结果

更好的写法：
- 做一段科技感 BGM，适合产品发布视频，30–60 秒，旁白友好

### 常见错误 6：把歌名写成一句文案
- 夏天吹来一阵风
- 热天里的小清凉

问题：
- 更像文案、歌词片段或社交标题
- 不像真正适合上架或播放列表展示的歌曲名

更好的写法：
- 夏风过境
- 小清凉
- 夏日慢拍

## 常见使用问题

### 1. 只给一个主题，可以吗？
可以。但如果主题太宽，比如“杭州”“夏天”“父亲”，skill 往往会继续确认：**你到底想表达什么**。

### 2. 我不会写风格怎么办？
没关系。你可以直接说你想要的感觉，也可以给一个喜欢的歌手或作品作为参考，系统会帮你抽象成风格特征，而不是做直接模仿。

### 3. 我不想填很多信息怎么办？
也没关系。这个 skill 默认是**少问、快收敛**，不是调查问卷型流程。

### 4. 它会直接模仿某个歌手吗？
不会。它会提炼风格特征、情绪、编曲质感和演唱气质，而不会输出复制型模仿内容。


## 如果你喜欢这种工作方式

- 它不会一上来问你十几个问题

- 它默认给你完整结果，而不是零散片段
- 它适合继续来回打磨，而不是一次性结束
